# Advantage One Interactive Portal - Frontend

This repository contains the `AIP_UI` frontend for the Advantage One Interactive Portal.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Install

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build

```bash
npm run build
```
